## Comforts — Full-Stack Product Management App

A full-stack web application built with Next.js (frontend) and Express.js + MongoDB (backend) for managing and displaying products.
Features include product creation, editing, deletion, user authentication, and detailed product views.


## Tech Stack

# Frontend
✔ Next.js 16.04 (App Router)
✔ React
✔ Tailwind CSS + DaisyUI
✔ NextAuth (optional)
✔ Vercel Deployment

# Backend
✔ Node.js
✔ Express.js
✔ MongoDB
✔ Vercel Deployment


# Features

✔ Add new products
✔ Edit products 
✔ Display product details by dynamic route
✔ Delete products
✔ Fully responsive UI
